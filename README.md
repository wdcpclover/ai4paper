# AI4Paper

**让天下没有难读的 Paper，让天下没有难做的科研。**

AI4Paper 是一款基于 AI 的 Zotero 插件，帮助科研人员高效管理、阅读和分析学术文献。

## 功能特性

🔍 **找文献** — AI速搜文献、领域新文监控、PDF智能识别

🤖 **读文献** — AI简记、AI精读、Ask PDF、思维导图、划词翻译

📝 **写综述** — 速览综述、深度综述、Research Gap 分析

🧹 **整理文献** — 智能去重、清理空壳、合并文件夹、自动归类

📊 **评估期刊** — SCIE/SSCI、JCR分区、中科院分区、影响因子、预警期刊

## 安装

1. 下载最新版 XPI 文件：[Releases](https://github.com/wdcpclover/ai4paper/releases)
2. 打开 Zotero → 工具 → 插件
3. 点击齿轮图标 → Install Add-on From File → 选择下载的 XPI 文件

支持 Zotero 7 / 8。
