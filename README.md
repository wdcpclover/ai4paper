# AI4Paper — Zotero 一站式 AI 科研平台

**让天下没有难读的 Paper，让天下没有难做的科研。**

AI4Paper 是 Zotero 插件，覆盖从找文献到写综述的全流程。所有 AI 功能基于真实文献，快但不糙。

支持 Zotero 7 / 8 | 官网：[ai4paper.pro](https://ai4paper.pro)

---

## 功能特性

### 🔍 找文献

- **AI 速搜文献** — 输入中文关键词，AI 自动翻译成多组英文变体，同时搜索多个学术数据库，结果按引用量排序，一键导入 Zotero
- **期刊分级** — 搜索结果自动标注中科院分区、JCR 分区（Q1-Q4）、影响因子、SCIE/SSCI/CSCD/CSSCI 收录情况
- **中文 PDF 识别** — 中文论文拖入 Zotero 自动识别标题、作者、期刊、摘要、关键词
- **领域追踪** — 设定研究领域，自动锁定高质量期刊，定期检索新论文自动导入
- **真假文献识别** — 粘贴参考文献列表，逐条比对权威数据库验证，防止引用 AI 编造的假论文

### 📖 读文献

- **AI 阅读辅助** — 打开 PDF 点击"阅读辅助"，问什么答什么，术语解释、公式拆解、方法对比
- **划词翻译** — 12 种翻译引擎内置（火山/Google/DeepL/有道/百度等），选中即翻译，侧边面板或悬浮浮窗
- **划词 AI 解释** — 选中文字弹出 AI 按钮，点击浮窗流式显示 AI 解读
- **思维导图** — 一键生成论文结构导图，支持 PNG / OPML 导出，可导入 XMind 等工具
- **AI 笔记** — 深度分析、简记、精读、对话、专业综述等多种笔记自动生成保存
- **笔记管理** — 所有 AI 笔记汇总在一个面板，按类型/时间/文件夹筛选，批量导出 Markdown
- **笔记同步** — Zotero 自带同步 或 AI4Paper 云端 AES-256 端到端加密同步

### 📁 管文献

- **PDF 一键整理** — 整个文件夹 PDF 导入，自动提取 DOI、查询元数据、重命名、归档
- **刷新元数据** — 一键补全标题、作者、摘要、DOI、被引量，英文标题自动翻译为中文
- **刷新期刊分区** — SCI / 中科院分区 / JCR / 影响因子，自定义列显示
- **AI 自动标签** — 92 条内置规则零成本秒打 + AI 智能中英双语标签
- **智能清理** — 重复文献 / 空条目 / 孤儿附件 / 失效附件 / 低质量论文，五类问题一次扫完
- **高级筛选** — 年份、分区、收录、标签等多条件组合筛选

### ✍️ 写综述

- **基础综述** — 选一批文献，三种深度：标题写综述（轻量）/ 摘要写综述（标准）/ 精读写综述（深度）
- **一键综述分析** — 输入关键词 → 自动搜文献 → 引文网络爬取 → 多轮 AI 筛选 → 生成 3000-5000 字专业综述报告
- **专业综述报告** — 综述引言 / 研究主题聚类 / 方法论对比 / 核心发现与矛盾 / Gap 分析 / 未来研究路线图，每个观点标注真实文献编号
- **迭代打磨** — 筛选出的核心文献可反复精选、换角度重新生成、追加新文献
- **文献矩阵** — 多篇论文自动生成对比表格（研究问题/方法/数据/结果/贡献/局限）

### 🛠️ 基础工具

- **下载附件（多源）** — 多源自动下载 PDF
- **导出 BibTeX / 导入 PDF** — LaTeX 引用导出、批量 PDF 导入
- **护眼模式** — 10 种预设护眼色，PDF 阅读更舒适
- **阅读追踪** — 自动记录每篇论文的阅读时长和进度
- **知识库问答** — 基于 RAG 的文献库问答，所有论文变成可检索的知识库

## 安装

1. 下载最新版 XPI 文件：[Releases](https://github.com/wdcpclover/ai4paper/releases)
2. 打开 Zotero → 工具 → 插件
3. 点击齿轮图标 → Install Add-on From File → 选择下载的 XPI 文件
4. 重启 Zotero，注册账号获得 **7 天全功能试用**

支持 Zotero 7 / 8。
