AI4Paper Scholar Privacy Policy

Last updated: March 28, 2026

Data Collection

AI4Paper Scholar does not collect, store, or transmit any personal data. The extension operates entirely on the client side for page content analysis.

Network Requests

The extension sends journal names extracted from academic web pages to the AI4Paper API (ai4paper.pro) to retrieve journal ranking information. Only journal names are transmitted; no user identifiers, browsing history, or personal information is sent.

Storage

The extension uses Chrome's storage API solely to save user preferences (API base URL). No personal data is stored.

Third-Party Services

The extension communicates only with ai4paper.pro to look up journal rankings. No data is shared with any other third party.
